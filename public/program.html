Program Yayasan

Program Beasiswa

Memberikan beasiswa kepada siswa yang berprestasi dan kurang mampu Sebanyak +/- 12 siswa telah tergabung dari penjuru jawa barat.

Program Kemanusiaan

Memberikan santunan kepada anak yang kurang prasejahtera menerima manfaat makanan sehat dari para donatur.

Program Kemanusiaan

Program berbagi setiap minggu pada hari jumat. Harapannya dapat memuliakan anak-anak yatim, dhuafa, terlantar dan disabilitas menerima manfaat makanan sehat dari para donatur..

Program Semarak Ramadhan

Memberikan kesadaran dan edukasi tentang pentingnya menjaga lingkungan dan mengurangi dampak buruknya.

Program LOGistik BUka dan SahUR

Program berbagi makanan/ logistik untuk sahur dan berbuka bagi para santri selama bulan Ramadhan dan Shaum Senin Kamis.

Program Tebar Hewan Qurban [ THR ]

Program penyebaran hewan qurban dalam bentuk hewan hidup atau distribusi daging qurban di wilayah minim qurban untuk anak-anak dan keluarga tidak mampu.
